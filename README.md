I created a music player app using vanilla HTML CSS and Javascript. Users can pause, play and skip songs, display song lyrics, create edit and delete playlist, search songs by title artist or album, upload their own songs, and navigate using app specific keyboard controls. 

While the playlist is on local storage so each user can make their own without accounts the songs are on a Supabase database. Each song has an ID number, title, artist, album, lyrics, and src. Because any user can update the same data set any songs uploaded can be seen by anyone. The functions work through six separate js files that all connect to index.html which connects to three css files. 

I went into the project with a much clearer plan than I have any of the others, and because I gave the AI a lot more from the beginning it matched more closely to what I had originally envisioned. It also asked me a lot more questions because of my more specific prompt so I was able to more clearly see how the AI processed imputed information, where I needed to guide it, and where it could figure things out.
